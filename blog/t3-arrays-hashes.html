The Power of Git

11/02/14

The Power of Git Version control is useful becuase it is a system that allows us to keep track of the changes to make to a file or a set of files over time. We are able to revert back to a previous version at any time, just incase the most recent changes aren't the exact way you like them. In our case, if our code isn't working properly, debugging properly, we can revert back to a previous version rather than pulling our hair out trying to figure out how many errors there were, or at least compare. This seperate version is called creating a branch, which is a copy of the master. Git is a version control system, that allows us to do what I wrote in the first paragrpah. Git enables us to do all of this from our terminal/command line, which is super useful to all users. It helps us track changes, and allows us to use commands like "git add" to help stage the change, "git commit" to commit a message to show what was changed, to help being organized. Being able to all this helps us keep track of what is done, and thus makes git important. Github allows us to store our files over the net, and can be shared publicly with others. It is used by a lot of people for collaboration and worked on by groups. It is THE social media site for programmers!

Array's and Hashes

11/16/14

Throughout the week, working with Ruby has definitely been a struggle. Definitely went more in depth into the language than we did studying for our interview with DBC. Went through a lot of ups and downs with the material and can't say i've fully understood everything, but definitely have a greater sense in the language now.
Some of the things we learned this week were Array's and Hashes which gets pretty useful in specific situations. Array's can hold objects of any data type. Each one of these elements can be referred to as the index of the array. The first element would start at 0. Starting at 0 is definitely weird since 1 is usually the starting point for.. well... most of everything.
For example an Array can consist of 5 different integers:

nums = [1 , 2, 3, 4, 5]

nums[0] would be 1

nums[1] would be 2 and so forth

We can also put strings in there :

names = [Mike, Jimmy, John, Ruby, Java]

names[0] would be Mike names[1] would be Jimmy and so forth.

Something else we learned for the week would be hashes. Hashes are a collection of key-value pairs. They are useful when storing the value within the key, so it may display when called.

For example:

fruits = {"apple" => red, "banana" => yellow, "lime" => green}

Definitely have to get more into Hashes, but we've definitely used arrays in a majority of our assignments this week. Looking forward to learning much more with this these coming up weeks!
