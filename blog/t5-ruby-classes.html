Classes

12/7/2014

Classes define what an object is, and what an object can do. To begin using a class we ned to define a class.

class Somename
end

We have just defined a class. When a class is defined, the first letter has to be capitalized or you will get an error.

Next up, we'll talk about some instances. An Instance is an object created from a class. An instance variable would have a @ in front of it. This Instance variable can be used throughout the class.

In this example i'll create a class Animal and have it output a noise.

class Animal << defining the class Animal
def setting_noise << defining setting_noise
@noise = "Woof" << setting instance variable
end
def making_noise << defining the method making_noise
@noise << instance variable
end
end

animal1 = Animal.new << driver code to run code
animal1.setting_noise << calling setting_noise method
puts animal1.make_noise << calling making_noise method
